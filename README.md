# City - smart


Plataforma de una ciudad inteligente desarrollado por Beestudents utilizando
tecnologias  para frontEND, UI/UX y acceso a algunas APIs publicas.

## Caracteristicas

- Login y registro de usuarios
- Dashboard Responsivo.
- Integracion con Firebase (Pasos evolutivos)
- Consumo de APIs publicas (clima , noticias, calidad del aire )

## Tecnologia utilizada (stack de desarrollo).


- HTML 5
- CSS 3
- Bootstrap 5
- Javascript
- Firebase
